
# 项目结构

--------

```txt
├── apps                        // 项目里的主骨架
|  └── web
├── docs                        // 项目文档/截图
├── packages                    // 项目的所有模块都在这里    
|  ├── tsdaodaobase             // 基础模块   （包含基础的聊天功能和一些常用的组件封装）
|  ├── tsdaodaocontacts         // 联系人模块 （通讯录）
|  ├── tsdaodaodatasource       // 数据源模块 （实现悟空IM相关的数据源）
|  └── tsdaodaologin            // 登录模块   （登录注册相关）
```
