
# 项目结构

--------

```txt
├── Modules                   // 模块入口，大部分业务逻辑都在这里，以后开发新模块也在这里
|  ├── WuKongBase             // 基础模块，其他模块都依赖这个模块，包含基础消息的UI定义，会话列表，消息列表和一些网络的封装等等
|  ├── WuKongContacts         // 联系人模块，包含好友、群组、黑名单等
|  ├── WuKongDataSource       // 数据源模块，给SDK提供数据源，包含用户信息，群组信息，会话信息等
|  └── WuKongLogin            // 登录模块，包含登录、注册、找回密码等
└── TangSengDaoDaoiOS         // 程序入口(包含项目启动页，项目Logo等等)

```
